# Piscine Aquamore - pubblicazione

Questa cartella contiene una copia indipendente dell'app turni per Piscine Aquamore.

## Cosa contiene

- Nome attivita preimpostato: Piscine Aquamore
- Reparti iniziali: Reception, Bagnini, Istruttori, Pulizie, Bar
- Luogo iniziale: Piscine Aquamore
- Dipendenti iniziali: nessuno
- Turni iniziali: nessuno
- Presenze, richieste e storico: vuoti

## GitHub

1. Crea un nuovo repository, per esempio `turni-aquamore`.
2. Carica tutti i file di questa cartella.
3. Fai commit.

## Render

1. Crea un nuovo Blueprint da quel repository.
2. Imposta una password manager diversa da Bar Flora con `MANAGER_PASSWORD`.
3. Verifica che il disco persistente si chiami `turni-aquamore-data`.
4. Apri l'app e configura i dipendenti reali da `Dipendenti`.

## Nota dati

Questa copia non deve condividere database, password, disco Render o repository con Bar Flora.
